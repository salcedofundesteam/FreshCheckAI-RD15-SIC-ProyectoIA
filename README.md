# FreshCheckAI-RD15-SIC-ProyectoIA

Este proyecto de inteligencia artificial se enfoca en la clasificación automática de frutas, según su estado de maduración. A través del análisis de imágenes, el modelo es capaz de identificar si una fruta está verde, madura o dañada.

La idea principal es facilitar el control de calidad en procesos agrícolas y comerciales, reduciendo errores humanos y optimizando el tiempo de selección. El sistema utiliza técnicas de visión por computadora y aprendizaje automático para reconocer patrones visuales como color, textura y posibles defectos en la fruta.
